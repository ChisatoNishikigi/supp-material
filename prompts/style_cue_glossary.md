# Style Cue Glossary

This file documents the style-specific cue glossary used to guide subsequent structured analysis.

## real_shot

- shadow direction inconsistency
- object identity mismatch within span
- duplicate/static frames
- abrupt time jump
- lighting flicker unnatural
- physics violation motion
- edge halo/over-sharpening
- compression-glitch segment

## rendered_3d

- style/texture tiling
- over-clean no sensor noise
- lighting/shadow physically off
- material specular unrealistic
- object geometry deformation
- time/discontinuity jump
- perfect edges changing
- physics mismatch collision

## animated_2d

- copy-paste frame reuse
- line art thickness inconsistency
- character attribute jump
- perspective error between frames
- impossible motion
- abrupt time jump
- color/flat shading mismatch
- narrative/logic break

## screen_recording

- UI state jump without action
- duplicate/static frames
- resolution/scale sudden change
- cursor teleport
- time text jump
- panel appears/disappears
- progress bar jump
- narrative step missing

## mixed_style

- style shift at cut
- identity mismatch across styles
- object state mismatch across styles
- lighting model jump
- narrative continuity break
- abrupt time jump
- motion discontinuity
- audio-visual desync
