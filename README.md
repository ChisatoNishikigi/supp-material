# Anonymous Supplementary Repository

This repository provides supplementary materials for an anonymous paper submission.

It is intended to support the claims and experimental settings described in the submission by providing:
- dataset metadata,
- source composition summaries,
- source links and access notes, and
- additional information on dataset usage in the experiments.

## Contents

The repository currently contains supplementary documentation related to the datasets used in the paper, including:
- additional dataset details,
- split usage notes,
- source composition of the mixed-source evaluation set,
- metadata tables, and
- access information for public sources.
