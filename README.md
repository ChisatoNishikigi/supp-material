# Anonymous Supplementary Repository

This repository provides supplementary materials for an anonymous paper submission.

It is intended to support the submission by providing:

- dataset metadata,
- source composition summaries,
- source links and access notes, and
- additional information on dataset usage.

## Current Scope

This repository is designed as a supplementary resource rather than a full public code release.

It currently focuses on dataset-related supplementary materials, including:

- additional dataset details,
- split usage notes,
- mixed-source dataset composition,
- metadata tables, and
- access information for public sources.

## Dataset Overview

The submission uses GenBuster-200K and a mixed-source evaluation set referred to as GenMix. 
Detailed dataset descriptions, split usage notes, and source composition statistics are provided in:

- `DATASET_DETAILS.md`
- `GENMIX_SOURCES.md`
- `metadata/`

## Data Access

This repository does not directly redistribute all video files used in the experiments. Instead, it provides metadata, source references, access notes, and supplementary descriptions for materials that are publicly available or subject to external access conditions. 

Please refer to `DATA_ACCESS.md` for details.

## Anonymity

This repository is prepared for double-blind review. Identifying information has been removed or anonymized where appropriate. Additional materials may be released in a fuller form after the review process, subject to data availability and licensing constraints.
