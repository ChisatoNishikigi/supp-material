# VideoCrafter2 Sample Videos

This directory contains a small illustrative subset of synthetic videos associated with **VideoCrafter2**.

These files are included only as example materials in the anonymous supplementary repository. They are intended to provide a small, source-specific sample for inspection and supplementary reference, rather than to serve as a full redistributed dataset.

## Notes

- This subset is limited in scale and is not intended to reproduce the full dataset setting used in the paper.
- The examples are included to provide a concrete source-level illustration for the supplementary repository.
- Please refer to the main repository documentation for dataset structure, source composition, and access notes.
